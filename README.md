# üéØ Focus Flow Pro

**Focus Flow Pro** is a high-performance productivity Chrome Extension designed to help users reclaim their focus. Unlike traditional blockers that just stop you from visiting a site, Focus Flow uses "Gentle Nudges" like grayscale, blurring, and transparency to discourage mindless scrolling.

## üöÄ Key Features
- **Custom Site Limits:** Set specific time limits (Hours, Minutes, Seconds) for any distracting website.
- **Deep Work Mode üî•:** A high-stakes mode that disables the "Snooze" button to force absolute focus.
- **Analytics Dashboard:** A dedicated full-page dashboard with interactive **Chart.js** visualizations.
  - **Line Chart:** Track your total wasted time trends over the last 30 days.
  - **Doughnut Chart:** See exactly which sites are stealing most of your time.
- **Dark Mode Support:** A beautiful, eye-friendly dark theme for late-night productivity sessions.
- **Accountability Grading:** Get a daily "Productivity Grade" (A+ to F) based on your performance.
- **CSV Export:** Download your rules and data for external time-tracking analysis.
